## Modo demostración

El sistema incluye un usuario de demostración con acceso de solo lectura.
Las operaciones de creación, edición y eliminación están bloqueadas por permisos a nivel API.

## Arquitectura

- Frontend: Vue 3 + Vite
- Backend: Django + Django REST Framework
- Base de datos: PostgreSQL
- Deploy backend: Render
- Deploy frontend: Netlify
